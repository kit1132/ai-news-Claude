# AI News Digest - 2026-04-17

木曜日。Claude Opus 4.7 GA リリース、Claude Design リサーチプレビュー、Claude Code Desktop 再設計 + Routines。OpenAI Codex がマルチエージェント + Desktop 制御で対抗。Gemini Mac ネイティブアプリ公開。

## Anthropic / Claude

- **Claude Opus 4.7 GA リリース（4/16）** — Anthropic 史上最強の一般提供モデル。SWE-bench Verified 87.6%、GPQA 94.2%。高解像度ビジョン対応（最大2576px / 3.75MP、従来の約3.3倍）。新トークナイザー（テキスト処理で1〜1.35倍のトークン消費）。タスクバジェット機能（トークン残量カウントダウンで作業優先度を自動調整）。Adaptive thinking が唯一の思考モードに。API価格据え置き（入力$5/出力$25 per M tokens）。Bedrock・Vertex・Foundry同時提供
  - https://www.anthropic.com/news/claude-opus-4-7

- **Claude Design リサーチプレビュー（4/17）** — 新プロダクト。テキストプロンプトからプロトタイプ、スライド、ワンページャー等のビジュアルを生成。画像・DOCX・PPTX・XLSXアップロード対応。Webキャプチャツールでサイト要素を取り込み可能。チームのデザインシステムを読み込んで一貫したスタイルを適用。Opus 4.7搭載。Pro/Max/Team/Enterprise向け
  - https://techcrunch.com/2026/04/17/anthropic-launches-claude-design-a-new-product-for-creating-quick-visuals/

- **Claude Code v2.1.111-112（4/16）** — Opus 4.7 `xhigh` エフォートレベル追加（`/effort` でスライダー操作）。Auto mode が Max 加入者向けに利用可能（`--enable-auto-mode` 不要に）。`/ultrareview`（クラウド並列マルチエージェントコードレビュー）。`/less-permission-prompts`（許可プロンプト削減スキル）。Auto テーマ（ターミナルのダーク/ライト自動追従）。PowerShell tool（Windows プログレッシブロールアウト）。Plan ファイルがプロンプト名ベースに。多数のバグ修正
  - https://code.claude.com/docs/en/changelog

- **Claude Code Desktop 再設計 + Routines リサーチプレビュー（4/14-15）** — デスクトップアプリを全面刷新。マルチセッションサイドバー、ドラッグ＆ドロップレイアウト、統合ターミナル、リビルドされた差分ビューア。サイドチャット（Cmd+;）で実行中タスクから分岐質問。Routines はスケジュール/API/Webhook の3種類。Anthropic Web インフラで実行されローカルマシン非依存。Pro: 5回/日、Max: 15回/日、Team/Enterprise: 25回/日
  - https://claude.com/blog/claude-code-desktop-redesign
  - https://claude.com/blog/introducing-routines-in-claude-code

- **Mythos 関連続報（4/16-17）** — Bloomberg 特集「Too Dangerous For Release」掲載。ホワイトハウスが連邦省庁へのMythosアクセス準備開始（OMBが保護措置設定中）。ECBが幹部との協議でMythos精査へ。ASAがSEC市場追跡DBへのリスクを警告。Bloomberg Live Q&A「Mythos Ushers in Dangerous New AI Era」開催（4/17）
  - https://www.bloomberg.com/news/features/2026-04-16/how-anthropic-discovered-mythos-ai-was-too-dangerous-for-release

## OpenAI

  - https://thetechportal.com/2026/04/17/openai-upgrades-codex-with-multi-agent-workflows-and-desktop-app-control-to-challenge-anthropics-claude-code

- **GPT-5.4-Cyber（4/14）** — 防御的サイバーセキュリティ特化のファインチューンモデル。バイナリリバースエンジニアリング対応。正当なセキュリティ作業に対する拒否境界を緩和。Trusted Access for Cyber（TAC）プログラムを数千人の検証済み個人・数百チームに拡大。限定的・段階的デプロイ
  - https://openai.com/index/scaling-trusted-access-for-cyber-defense/

- **GPT-5.5 "Spud" リリース近づく** — Polymarket 4/23 リリース確率が85.5%に上昇（前日78%）。GPT-5.5 か GPT-6 の名称は性能ジャンプ次第で未確定のまま

  - https://gemini.google/mac/
  - https://9to5google.com/2026/04/15/gemini-app-mac/

- **Google Meet CarPlay 対応 + 音声翻訳モバイル展開** — Meet が Apple CarPlay で利用可能に（音声のみ、映像なし）。リアルタイム音声翻訳がAndroid/iOSアプリに展開中（英語⇔スペイン語/フランス語/ドイツ語/ポルトガル語/イタリア語）。Android Auto 版も近日
  - https://workspaceupdates.googleblog.com/2026/04/google-meet-is-now-available-on-carplay.html

## Cursor

- **Cursor Canvases（4/16）** — エージェントがインタラクティブなビジュアルインターフェースを生成する新機能。PRレビュー、データ分析、リサーチ結果をダッシュボード・図表・To-doリスト等で可視化。React ベースのコンポーネント。Agents Window のサイドパネルに永続表示。スキルでカスタムキャンバス定義可能（例: Docs Canvas でリポジトリアーキテクチャ図を自動生成）。Cursor 3.1
  - https://cursor.com/changelog
