## Anthropic / Claude

### Claude Opus 4.7 リリース（4/16）
Anthropicの最新フラッグシップモデル。コーディングベンチマーク13%向上、CursorBenchで4.6比12ポイント改善。高解像度ビジョン（最大2576px / 3.75MP、従来1568px / 1.15MP）、タスクバジェット（エージェントのトークン消費上限管理）、新しいxhighエフォートレベル、新トークナイザー（テキストで最大35%トークン増の可能性）を導入。API料金は4.6と同じ（$5/$25 per 1M tokens）。API、Bedrock、Vertex AI、Microsoft Foundryで提供。
- https://www.anthropic.com/news/claude-opus-4-7

### Claude Design by Anthropic Labs（4/17）
Opus 4.7ベースのビジュアルデザインツール。テキストプロンプト、画像、DOCX/PPTX/XLSXアップロード、Webキャプチャからデザイン・プロトタイプ・スライドを生成。チームのデザインシステムを自動適用可能。Pro/Max/Team/Enterprise向けリサーチプレビュー。
- https://www.anthropic.com/news/claude-design-anthropic-labs

### Google → Anthropic 最大$40B投資（4/24）
Googleが$10Bを即時投資（$350B評価額）、Anthropicがパフォーマンス目標を達成すれば追加$30B。Google既存投資は$3B超（14%株式保有）。
- https://www.cnbc.com/2026/04/24/google-to-invest-up-to-40-billion-in-anthropic-as-search-giant-spreads-its-ai-bets.html

### Amazon → Anthropic 最大$25B追加投資 + 5GW計算資源（4/20）
10年間でAWS技術に$100B以上コミット。Trainium2/3/4および将来世代のカスタムシリコン、数千万Graviton CPUコアを含む。2026年末までにTrainium2+3合計約1GW稼働予定。Claude PlatformがAWS内から直接アクセス可能に。
- https://www.anthropic.com/news/anthropic-amazon-compute

### Anthropic ARR $30B突破
$9B（2025年末）→ $30B（2026年4月）の急成長。OpenAIのARR $25Bを初めて超過。年間$1M以上支出の法人顧客1,000社以上（2月比で倍増）。
- https://www.bloomberg.com/news/articles/2026-04-06/broadcom-confirms-deal-to-ship-google-tpu-chips-to-anthropic

### Claude Code v2.1.113〜v2.1.119（4/17〜4/23）
主要な変更点:
- **v2.1.113（4/17）**: ネイティブバイナリ化（JavaScriptバンドルから移行）、`sandbox.network.deniedDomains`設定、セキュリティ強化（`env`/`sudo`ラッパー検出、`find -exec`/-delete自動承認防止）
- **v2.1.116（4/20）**: `/resume`最大67%高速化、フルスクリーンスクロール改善、Opus 4.6/Sonnet 4.6デフォルトeffort `high`化
- **v2.1.117（4/22）**: フォークサブエージェント、エージェントfrontmatter `mcpServers`対応、Opus 4.7の1Mコンテキスト正常認識修正
- **v2.1.118（4/23）**: Vim visual mode（`v`/`V`）、カスタムテーマ作成、`type: "mcp_tool"`フック、`/cost`と`/stats`を`/usage`に統合
- **v2.1.119（4/23）**: `/config`設定永続化、`--from-pr`がGitLab/Bitbucket/GHE対応、PowerShellコマンド自動承認、`PostToolUse`に`duration_ms`追加
- https://code.claude.com/docs/en/changelog

## OpenAI

### GPT-5.5リリース（4/23〜24）
OpenAI最新フラッグシップ。コーディング、コンピュータ操作、リサーチ、データ分析に特化。1Mコンテキスト。Plus/Pro/Business/Enterprise向けChatGPT・Codexで展開中。API料金: GPT-5.5 $5/$30、GPT-5.5 Pro $30/$180（per 1M tokens）。Batch/Flexで50%割引。実効コストはGPT-5.4比約20%増。
- https://openai.com/index/introducing-gpt-5-5/

### GPT-Rosalind（4/16）
ライフサイエンス特化の推論モデル。創薬・タンパク質工学・ゲノミクスに最適化。Amgen、Moderna、Thermo Fisherなどが早期採用。Trusted Accessプログラム経由、米国Enterprise限定リサーチプレビュー。
- https://openai.com/index/introducing-gpt-rosalind/

### Codex Labs（4/21）
エンタープライズ向けCodex導入支援プログラム。Accenture、Capgemini、Cognizant、Infosys、PwC、TCSなど大手SIがパートナー。

### Agents SDK アップデート（4/15）
ファイル操作・コマンド実行・コード編集・長期タスクをサンドボックス内で実行可能に。構成可能メモリ、Codex風ファイルシステムツール、標準化インテグレーション追加。
- https://openai.com/index/the-next-evolution-of-the-agents-sdk/

### Sora アプリ版停止（本日 4/26）
Soraウェブ/アプリ版が本日停止。API版は9/24まで。運用コスト$1M/日、ユーザー100万→50万未満に減少が背景。Disneyの$1B提携も破談。
