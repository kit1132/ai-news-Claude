# AI News Digest - 2026-04-02

## Claude Code / Anthropic

### Claude Code v2.1.89 リリース（4/1）
- **Deferred permission decisions**: `PreToolUse` フックで `defer` を返せるように。ヘッドレスセッションがツールコールで一時停止し、`-p --resume` で再評価可能
- **フリッカーフリーレンダリング**: `CLAUDE_CODE_NO_FLICKER=1` 環境変数で alt-screen レンダリング + 仮想スクロールバック
- **PermissionDenied フック**: auto mode 分類器の拒否後に発火。`{retry: true}` でモデルにリトライを指示可能
- **名前付きサブエージェント**: `@` メンションのタイプアヘッドに表示
- **MCP 非ブロッキング接続**: `MCP_CONNECTION_NONBLOCKING=true` で `-p` モード時の接続タイムアウトを5秒に
- `/buddy` コマンド（エイプリルフールのイースターエッグ — コーディング中に小さな生き物がついてくる）
- 50以上のバグ修正（ボイスモード、ファイル編集、メモリリーク、LSPクラッシュ等）
- https://code.claude.com/docs/en/changelog

### Claude Code ソースコード流出事件（3/31〜4/1）
- v2.1.88 の npm パッケージに 59.8MB のソースマップファイル（.map）が誤って含まれ、約512,000行の TypeScript コードが公開
- **原因**: Anthropic が2025年末に買収した Bun のバグ（issue #28001）により、本番ビルドでソースマップが含まれた
- **発覚**: Solayer Labs インターンの Chaofan Shou 氏が X で発見を共有
- **流出内容**: 内部コードネーム（Capybara = Claude 4.6系、Fennec = Opus 4.6、Numbat = テスト中）、「KAIROS」フィーチャーフラグ（常時バックグラウンド動作のデーモンモード）、「Proactive mode」（プロンプトなしで動作）
- **Anthropic の声明**: 「顧客データや認証情報の漏洩なし。リリースパッケージングの人為的ミス」
- **注意**: 同時期に axios npm パッケージへのサプライチェーン攻撃も発生。3/31 00:21〜03:29 UTC に Claude Code をインストール/更新した場合、悪意ある axios（RAT入り）を取得した可能性あり
- https://venturebeat.com/technology/claude-codes-source-code-appears-to-have-leaked-heres-what-we-know
- https://www.bleepingcomputer.com/news/artificial-intelligence/claude-code-source-code-accidentally-leaked-in-npm-package/
