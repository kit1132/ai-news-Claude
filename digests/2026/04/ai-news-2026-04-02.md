### Claude Code トークン消費問題が深刻化
- Anthropic が「usage limits に想定より遥かに早く到達している。チームの最優先事項」と認める
- Max 5 プラン（$100/月）で**19分で枠を使い切る**事例が報告
- **バグ1（Sentinel String Replacement）**: カスタム Bun フォーク内のネイティブ層が JSON リクエストボディを改変し、会話履歴に特定文字列があるとキャッシュプレフィックスが破壊 → コスト10〜20倍に膨張
- **バグ2（Session Resume Token Drain）**: セッション再開時にユーザー入力なしで 652,069 出力トークンが生成される事例
- **回避策**: 旧バージョン（v2.1.34等）へのダウングレード、Node.js 経由での実行、未使用 MCP コネクタの切断
- https://www.theregister.com/2026/03/31/anthropic_claude_code_limits/
- https://dev.to/fillip_kosorukov/claude-code-is-silently-burning-10-20x-your-token-budget-heres-the-fix-4mpk
