# AI News Digest — 2026-03-27

## Claude Code / Anthropic

### Claude Code v2.1.84（2026-03-26）
- **PowerShell ツール**（Windows向けopt-inプレビュー）追加
- **環境変数によるモデル機能検出**: `ANTHROPIC_DEFAULT_{OPUS,SONNET,HAIKU}_MODEL_SUPPORTS`、`_MODEL_NAME`、`_DESCRIPTION`
- **ストリーミングアイドルタイムアウト**: `CLAUDE_STREAM_IDLE_TIMEOUT_MS` で設定可能（デフォルト90秒）
- **新フック**: `TaskCreated` フック追加、`WorktreeCreate` が `type: "http"` をサポート
- **MCP改善**: ツール説明文が2KB上限に。ローカル設定がclaude.aiコネクタより優先されるよう重複排除
- **トークン表示**: 1M以上の値が「1.5m」形式で表示（「1512.6k」→「1.5m」）
- **rules/skillsパス**: `paths:` フロントマターがYAMLリストのglobに対応
- **その他**: バックグラウンドbashタスクの対話プロンプト検知（45秒後通知）、Voice改善、起動パフォーマンス約30ms改善
- ソース: https://code.claude.com/docs/en/changelog
