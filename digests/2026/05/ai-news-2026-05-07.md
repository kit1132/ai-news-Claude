### Claude Code v2.1.131 / v2.1.129（5/6） ※新規
- **v2.1.131**: VS Code 拡張の Windows 起動バグ修正（バンドル SDK の `createRequire` polyfill）、Mantle エンドポイント `x-api-key` ヘッダ欠落修正
- **v2.1.129**: `--plugin-url <url>` で zip プラグインを URL 取得、`CLAUDE_CODE_FORCE_SYNC_OUTPUT=1`（Emacs `eat` 等向け）、Homebrew/WinGet 用 `CLAUDE_CODE_PACKAGE_MANAGER_AUTO_UPDATE`、プラグインマニフェスト `themes`/`monitors` を `experimental` 配下に移動（top-level も警告付きで動作）、Gateway `/v1/models` 探索を `CLAUDE_CODE_ENABLE_GATEWAY_MODEL_DISCOVERY=1` の opt-in に戻す（v2.1.126〜128 で自動化されていた）、Ctrl+R 履歴ピッカーを全プロジェクト横断に戻す（Ctrl+S で現在プロジェクト/セッションに絞込）、3rd party デプロイ（Bedrock/Vertex/Foundry/`ANTHROPIC_BASE_URL`）でファースト パーティ向けスピナー Tip 抑制、`skillOverrides` 設定有効化、OTel `claude_code.pull_request.count` が MCP 経由 PR/MR もカウント、Policy refusal エラーに API Request ID 付与
- ソース: [Claude Code Changelog](https://code.claude.com/docs/en/changelog)

### Claude Mythos Preview の能力詳細が公開（5/5 red.anthropic.com） ※新規
- **全主要 OS／全主要ブラウザでゼロデイを発見・エクスプロイト可能**。発見済み脆弱性の99%超が未パッチ
- 例: 4つの脆弱性をチェーンしたブラウザエクスプロイト（renderer + OS sandbox エスケープ込みの JIT heap spray）、レースコンディション/KASLR バイパスによる Linux 等の自律的ローカル権限昇格
- 最古は **27年前の OpenBSD バグ**（既パッチ）
- 一般公開せず、Project Glasswing 経由でサイバーセキュリティパートナーに限定提供（重要インフラのパッチ用）。Dario Amodei は「中国系AIが追いつくまでの 6〜12ヶ月で世界の脆弱性をパッチする猶予」と発言
- ソース: [Claude Mythos Preview](https://red.anthropic.com/2026/mythos-preview/), [CNBC](https://www.cnbc.com/2026/05/05/anthropic-ceo-cyber-moment-of-danger-mythos-vulnerabilities.html)
