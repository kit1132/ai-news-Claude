## 最優先

### Anthropic × Gates Foundation：$200M パートナーシップ（5/13）
- 4年で **$200M**（助成金 + Claude 利用クレジット + 技術支援）。グローバルヘルス、ライフサイエンス、教育、経済モビリティの4領域
- ヘルス: 低中所得国の医療アクセス改善（約46億人対象）、ポリオ・HPV・eclampsia/preeclampsia 等の見落とされがちな疾患の研究加速、新ワクチン・治療開発
- 教育: 米国 K-12 向け evidence-based チューター + キャリア指導、サブサハラアフリカ・インドで識字/算数支援アプリ
- 経済モビリティ: 小規模農業（~20億人）向け Claude の農業特化改善 + データセット公開
- Anthropic にとって Pentagon との $200M 契約（4月）に続く非市場領域での大型コミット

### OpenAI Codex：ChatGPT モバイルアプリで提供開始（5/14）
- iOS / Android プレビュー。**全プラン（Free / Go 含む）対応**、全サポート地域
- ローカル Mac / リモート開発環境にスマホからアタッチ。アクティブタスクスレッド切替、コマンド承認、モデル切替、新規タスク投入、ライブ状態確認
- 現状は **macOS 対象**（Windows サポートは「coming soon」）

### GitHub Copilot App：テクニカルプレビュー公開（5/14）
- **GitHub-native デスクトップアプリ**（Windows / macOS / Linux）。agentic 開発をエージェント単位で隔離・操舵・PR レビューまで一気通貫
- Issue / PR / プロンプト / 過去セッション からセッション開始。各セッションは独立した **git worktree + branch + task state** を持ち、同一リポへの並列実行に対応
- 提供: Copilot Pro / Pro+ は早期アクセス登録可、Business / Enterprise は順次展開
- Copilot CLI から「ターミナル中心」、Copilot App で「デスクトップ中心」へラインアップ拡張。Claude Code Desktop / OpenAI Codex App との直接競合
- ソース: [GitHub Changelog](https://github.blog/changelog/2026-05-14-github-copilot-app-is-now-available-in-technical-preview/) / [GitHub Blog 日本語](https://github.blog/jp/2026-05-15-github-copilot-app-is-now-available-in-technical-preview/) / [Neowin](https://www.neowin.net/news/microsoft-launches-github-copilot-app-to-supercharge-agentic-development/) / [DevOps Journal](https://devopsjournal.io/blog/2026/05/14/github-copilot-app)

## 高優先

### Notion Developer Platform 公開（5/13）— External Agent API で Claude Code/Cursor/Codex を Notion 内に取り込み
- **Workers**: Notion ホスト型ランタイム。`ntn` CLI から deploy。ベータ中無料、8/11 から Notion クレジット消費
- **External Agent API**: 既存エージェントを Notion ワークスペース参加者として接続。**ローンチ時対応: Claude Code / Cursor / Codex / Decagon**（順次追加）
- **Database Sync**: Salesforce / Zendesk / Postgres など任意の API データソースを Notion DB に同期
- **Notion CLI（ntn）**: ワンタイム認証で読み書き・Workers 管理・ワークフロー自動化を CLI/IDE から
- 「Notion をエージェントのコントロールルーム化」する戦略。Notion 内ドキュメント/DB が AI エージェントの状態管理基盤になる
- ソース: [Notion 公式リリース](https://www.notion.com/releases/2026-05-13) / [Notion Blog](https://www.notion.com/blog/introducing-developer-platform) / [TechCrunch](https://techcrunch.com/2026/05/13/notion-just-turned-its-workspace-into-a-hub-for-ai-agents/) / [Dataconomy](https://dataconomy.com/2026/05/14/notion-launches-developer-platform-for-ai-workflows-and-agents/)
