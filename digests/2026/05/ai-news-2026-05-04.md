## 開発ツール

### Cloudflare Agents が Stripe Projects 経由で自律プロビジョニング（4/30、5/2 解説記事続報）
- エージェントが Cloudflare アカウント作成・ドメイン購入・有料サブスク開始・本番デプロイまで人間介在なしに完遂可能に
- **プロトコル3層**: ① REST/JSON catalog でサービス discovery → ② OAuth + identity attestation で権限委譲 → ③ Stripe payment token を Provider に渡し決済（生のカード番号はエージェントに非開示）
- デフォルト spending cap は **provider あたり $100/月**、CLIから複数サービス（Cloudflare / Supabase / Hugging Face / Twilio / AgentMail 他）を一括 provisioning 可能。Stripe Projects はオープンベータ
- ソース: <https://blog.cloudflare.com/agents-stripe-projects/> / <https://developers.cloudflare.com/agents/agentic-payments/>
