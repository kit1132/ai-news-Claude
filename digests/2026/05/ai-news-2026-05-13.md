## Anthropic / Claude

### Claude Code v2.1.139（5/11） ※新規
- **Agent View（Research Preview）**: `claude agents` で全 Claude Code セッション（running / blocked-on-you / done）を単一リストで一覧表示。複数並列セッション運用向けのコマンドセンター
- **`/goal` コマンド**: 完了条件（rubric / 自然言語）を設定すると、interactive / `-p` / Remote Control いずれのモードでも条件が満たされるまでターンをまたいで作業継続。経過時間・ターン数・トークン使用量をオーバーレイ表示
- **`/scroll-speed` コマンド**: マウスホイールのスクロール速度をライブプレビュー付きで調整
- **`claude project purge [path]`**: トランスクリプト・タスク・ファイル履歴・設定エントリを含む Claude Code プロジェクト状態を完全削除。`--dry-run` / `-y` / `--interactive` / `--all` 対応
- **Plugin Details 強化**: `claude plugin details <name>` でコンポーネント一覧とセッションあたりトークンコスト試算を表示
- **トランスクリプト操作**: `?` でキーボードショートカット表示、`{` / `}` でユーザープロンプト間ジャンプ、`v` でショートカットパネルトグル
- **Hook 強化**: exec form の `args: string[]`、`PostToolUse` hook の `continueOnBlock` 設定追加
- **MCP**: stdio サーバへ `CLAUDE_PROJECT_DIR` 注入、`.mcp.json` 編集時の再接続改善、エラーメッセージ改善
- v2.1.137（5/9）VSCode Windows 拡張起動修正のみ、v2.1.138（5/9）内部修正のみ
- ソース: [Claude Code Changelog](https://code.claude.com/docs/en/changelog)

- ソース: [OpenAI 公式](https://openai.com/index/openai-launches-the-deployment-company/), [Axios](https://www.axios.com/2026/05/11/openai-deployco-private-equity), [PYMNTS](https://www.pymnts.com/news/artificial-intelligence/2026/openai-launches-4-billion-dollar-company-accelerate-enterprise-ai-adoption/)

### ChatGPT Trusted Contact / Memory Sources 段階展開 ※新規
- **Trusted Contact**: 自殺関連の深刻な安全懸念を自動システムと専門レビュアーが検知した場合、ユーザーが事前指定した連絡先へ通知できるオプション機能。個人プランのみ（Business/Enterprise/Edu は対象外）、対応リージョン段階展開
- **Memory Sources**: 全コンシューマプランで「どの記憶（過去チャット・保存メモリ・Custom Instructions）が今の回答の元になったか」を Sources アイコンから確認・編集可能に。Plus/Pro は接続済み Gmail メール・ライブラリ内ファイルも参照可能。Web 先行、モバイル順次
- ソース: [ChatGPT Release Notes](https://help.openai.com/en/articles/6825453-chatgpt-release-notes), [AlternativeTo — Trusted Contact](https://alternativeto.net/news/2026/5/chatgpt-launches-trusted-contact-feature-for-user-safety-support/), [LearnPrompting — Memory Improvements](https://learnprompting.org/blog/chatgpt_memory_update)
