## 開発ツール（Cursor / Devin / GitHub Copilot CLI）

### Cursor 3.3（5/6-7）— Parallel Agents / PR Review 刷新 / Explore subagent 制御 ※新規（5/8 digest 漏れ補完）
- **Parallel Agent Execution（"Build in Parallel"）**: プランの独立部分を識別し、**async subagents で並列実行**。直列実行より大幅高速化
- **PR Review 体験刷新**:
  - Reviews タブ: インラインレビュースレッド + PR 全体コメント
  - Commits タブ: コミット履歴フォーカス表示
  - Changes タブ: ファイルツリー + 大型 PR 用 changes picker
  - 変更を **PR に分割**するビルトインアクション、skills のピン留め
- **Quick-action pills**: 共通ワークフロー用のクイックアクション
- **Explore subagent 制御**（settings から）: 専用モデル指定 / 親エージェント継承 / 完全無効化を選択
- **総称モデル名**（例: `model: opus`）で subagent が常に最新 Opus を使うよう指定可能
- **Context usage breakdown**: rules / skills / MCPs / subagents 別に context 消費を内訳表示。コンテキスト枯渇のデバッグに直結
- ソース: [Cursor Changelog](https://cursor.com/changelog), [MCP Apps and Team Marketplaces (2.6 page)](https://cursor.com/changelog/2-6)

### Devin Review API + Full Desktop Testing + v3 API GA（5/8） ※新規
- **Devin Review API**: REST API 経由で **Devin Review をプログラム的にトリガ**可能。CI/CD パイプラインへ組み込みやすく
- **Full Desktop Testing**: computer use ベースの **エンドツーエンド E2E テスト**。Linux で動作する任意のデスクトップアプリで、Devin が PR 上で QA を申請 → 承認すると実機操作の編集動画が返却
- **v3 API が GA に**: **RBAC（role-based access control）**、セッションアトリビューション、新機能。v1 / v2 レガシー API は将来廃止予定
- 追加: playbook 管理改善、enterprise 向け review permissions enforcement、integration ページ刷新
- ソース: [Devin Release Notes 2026](https://docs.devin.ai/release-notes/2026), [Devin Docs](https://docs.devin.ai/release-notes)
